h3. Short URLs are annoying. Expand them.

pre. % gem install expander-0.0.1.gem
Successfully installed expander-0.0.1
1 gem installed
% irb -rubygems
> require 'expander'
 => true 
>   s = "some stuff http://t.co/xCCP4e8R more stuff #hello"
 => "some stuff http://t.co/xCCP4e8R more stuff #hello" 
> s.expand_urls
 => "some stuff http://guides.rubygems.org/make-your-own-gem/ more stuff #hello"
